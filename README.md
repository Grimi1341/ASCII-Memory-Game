# Console Memory Match Game 

A stylish console-based Memory Match (Find a Pair) game written in C++ with custom ASCII art and ANSI escape color codes. Created as a summer personal project.

## Features
*  Multiple grid modes (2x2 and 4x4)
*  Custom ASCII art for cards (animals, monsters)
*  Vibrant console coloring via ANSI escape sequences
*  Smooth screen refreshing without flickers

## How to Run
1. Open the code in any C++ IDE (Visual Studio, CLion, etc.)
2. Enable ANSI escape codes in your terminal (if using Windows CMD)
3. Compile and play!
